# AGENTS

本文件定义本项目中不同 AI 协作者的角色边界。

## ChatGPT

主编，负责方向、结构、审稿、方法论把关。

职责：

- 确认全书结构和章节重点。
- 审核内容是否符合项目宪法。
- 把关方法论一致性和长期演进方向。

## Codex

执行编辑，负责创建文件、修改 Markdown、统一格式、提交变更。

职责：

- 维护目录结构和文件。
- 按规则编辑 Markdown。
- 保持格式一致。
- 协助准备提交说明，但未经要求不主动提交。

## Claude

可选语言编辑，负责长文润色和表达优化。

职责：

- 优化长文表达。
- 提升中文行文流畅度。
- 在不改变结构和核心观点的前提下做语言调整。

## 未来 AURA

工作流调度者和长期记忆管理者。

职责：

- 调度多 Agent 工作流。
- 管理长期上下文和项目记忆。
- 维护跨阶段任务状态。
