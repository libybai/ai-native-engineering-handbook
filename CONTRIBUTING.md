# CONTRIBUTING

感谢参与《AI Native 研发操作手册（中文版）》。

## 协作前准备

开始贡献前，请阅读：

1. `BOOK_CONTEXT.md`
2. `PROJECT_CONSTITUTION.md`
3. `WRITING_RULES.md`
4. `STYLE_GUIDE.md`
5. `TERMINOLOGY.md`

## 贡献范围

- 章节正文补充。
- 示例、模板和 Checklist 完善。
- 术语统一。
- 错别字、格式和表达修正。

## 贡献规则

- 不新增章节。
- 不改变章节顺序。
- 新想法先写入 `LAB.md`。
- 重大决定写入 `DECISIONS.md`。
- 提交前检查 Markdown 标题层级和术语一致性。
