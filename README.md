# AI Native 研发操作手册（中文版）

这是一本以 Book as Code 方式维护的《AI Native 研发操作手册（中文版）》。

本项目的目标是建立一套长期、可实践、与具体 AI 工具无关的软件研发体系，帮助团队把 AI 协作纳入日常研发流程，而不是把它当作一次性的提效技巧。

## 项目定位

- 面向软件研发团队、产品团队、设计团队和 AI 协作者。
- 关注工作流、标准、上下文管理、协作机制和可持续演进。
- 优先沉淀可复用实践，不做空泛理论研究。
- 不绑定具体 AI 工具、模型或厂商。

## 维护方式

本书采用 Book as Code 方式维护：

- 章节以 Markdown 文件管理。
- 结构、术语、风格和协作规则由根目录规范文件约束。
- 新想法先进入 `LAB.md`。
- 重大决定记录到 `DECISIONS.md`。
- 所有 AI 协作者必须先阅读项目上下文和写作规则。

## 目录结构

正文位于 `chapters/`，按五个部分组织。模板、示例和图片分别放入 `templates/`、`examples/`、`images/`。

## AI 协作入口

开始协作前，请依次阅读：

1. `BOOK_CONTEXT.md`
2. `PROJECT_CONSTITUTION.md`
3. `WRITING_RULES.md`
4. `STYLE_GUIDE.md`
5. `AGENTS.md`
