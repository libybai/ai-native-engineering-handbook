# TERMINOLOGY

本文件维护全书统一术语。

## 术语表

| 术语 | 定义 | 使用说明 |
| --- | --- | --- |
| AI Native | 将 AI 协作作为研发体系原生组成部分的工作方式。 | 避免仅解释为“使用 AI 工具”。 |
| Book as Code | 使用代码仓库、版本控制和文本文件维护书籍内容的方法。 | 本项目的基本维护方式。 |
| Context | AI 协作所需的背景、约束、状态和目标信息。 | 中文可写作“上下文”。 |
| Agent | 在特定角色和规则下执行任务的 AI 协作者。 | 保持中英文一致。 |

## 维护规则

- 新术语首次进入正文前，应先补充到本文件。
- 同一术语在全书中保持同一写法。
- 对外部工具名保持原名，但正文不围绕特定工具展开。
